# Documentation Templates

Esta pasta contém modelos de documentação reutilizáveis.

Esses arquivos **não são documentação do projeto**.

Eles são **templates** usados para gerar documentos dentro da pasta:

docs/

---

# Como usar

Quando for necessário criar documentação:

1. Criar a pasta `docs/` na raiz do projeto (se não existir)

2. Copiar o template correspondente desta pasta

3. Criar o documento final dentro de `docs/`

Exemplo:

templates/architecture-template.md
↓
docs/architecture.md

templates/workflow-template.md
↓
docs/workflow.md

templates/context-log-template.md
↓
docs/context-log.md
