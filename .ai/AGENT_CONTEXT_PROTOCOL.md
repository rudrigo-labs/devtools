# AGENT CONTEXT PROTOCOL

Define como a IA reconstrui o contexto do projeto.

Ordem de leitura:

1) KICKOFF.md
2) AGENT_EXECUTION_RULES.md
3) DOCUMENTATION_RULES.md
4) PROJECT_RULES.md
5) docs/context-log.md
6) docs/architecture.md (se existir)

Apos carregar os arquivos o agente deve:

- confirmar que o contexto foi carregado
- nao executar nada automaticamente
- aguardar instrucoes
