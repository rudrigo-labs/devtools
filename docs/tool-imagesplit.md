# DevTools.ImageSplit - Como usar (CLI)

## Objetivo

Separar componentes de uma imagem com fundo transparente.

## Passos

1. Selecione `Image Split`.
2. Informe o arquivo de imagem.
3. (Opcional) Ajuste output, base e extensao.
4. (Opcional) Configure parametros avancados.

## Entradas

- Arquivo de imagem (obrigatorio)
- Pasta de saida (opcional)
- Base do nome (opcional)
- Extensao (opcional)
- Alpha threshold (opcional)
- Start index (opcional)
- Sobrescrever (s/n)
- Min largura/altura (opcional)

## Saida

- Caminhos dos arquivos gerados
- Total de componentes
