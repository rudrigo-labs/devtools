# DevTools.Snapshot - Como usar (CLI)

## Objetivo

Gerar snapshots (TXT, JSON ou HTML) de uma pasta/projeto.

## Passos

1. Selecione `Snapshot` no menu.
2. Informe a pasta raiz.
3. Escolha o formato (ou todos).
4. (Opcional) Informe max KB por arquivo e pastas a ignorar.

## Entradas

- Pasta raiz (obrigatorio)
- Saida base (opcional)
- Formato (TXT, JSON achatado, JSON recursivo, HTML, todos)
- Max KB por arquivo (opcional)
- Ignorar pastas (opcional)

## Saida

- Artefatos gerados e caminho da saida base.

## Observacoes

- Se "Todos" for escolhido, o CLI gera as 4 saidas.
