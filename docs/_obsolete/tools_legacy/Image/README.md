# Image

**Overview**
Splits an image into components based on transparency (sprite splitter).

**Usage**
- CLI key: `split`
- Inputs: image path, optional output folder, base name, extension
- Advanced options: alpha threshold, start index, overwrite, min width, min height

**Configuration**
- No config file. All options are set in the CLI prompts.

**Output**
- Writes extracted components to the output folder

**Logs**
- Errors only: `%AppData%/DevTools/logs/split-YYYYMMDD.log`
