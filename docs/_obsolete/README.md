# Arquivos Obsoletos

Este diretorio armazena documentos legados que nao sao mais a fonte oficial.

## Subpastas

- `legacy_context/`: notas de contexto temporarias e templates antigos.
- `legacy_plans/`: planos substituidos por estrategias mais recentes.
- `legacy_tool_docs/`: guias antigos por ferramenta, consolidados em `docs/tools/`.
- `legacy_guides/`: guias gerais substituidos por versoes canonicas.

## Politica

- Nao editar arquivos aqui, exceto para referencia historica.
- Novas atualizacoes devem ser feitas apenas nos documentos ativos em `docs/`.
