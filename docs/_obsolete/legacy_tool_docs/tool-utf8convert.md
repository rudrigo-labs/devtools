# DevTools.Utf8Convert - Como usar (CLI)

## Objetivo

Converter arquivos de texto para UTF-8 e corrigir acentos.

## Passos

1. Selecione `Utf8 Convert`.
2. Informe pasta raiz.
3. Configure recursivo, dry-run, backup e BOM.

## Entradas

- Pasta raiz (obrigatorio)
- Recursivo (s/n)
- Dry-run (s/n)
- Criar backup (s/n)
- Gerar BOM (s/n)
- Includes (globs) (opcional)
- Excludes (globs) (opcional)

## Saida

- Arquivos escaneados
- Convertidos
- Ja UTF8
- Binarios ignorados
- Excluidos
- Erros
