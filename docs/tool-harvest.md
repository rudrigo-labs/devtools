# DevTools.Harvest - Como usar (CLI)

## Objetivo

Mapear um projeto e listar arquivos candidatos a serem mais importantes, com base em dependencias e regras de score.

## Passos

1. Abra o CLI e selecione `Harvest`.
2. Informe a pasta raiz do projeto.
3. (Opcional) Informe um arquivo de configuracao.
4. (Opcional) Informe `MinScore` e `Top N`.

## Entradas

- Pasta raiz (obrigatorio)
- Config (opcional)
- MinScore (opcional)
- Top N (opcional)

## Saida

- Total de arquivos analisados
- Total de arquivos pontuados
- Lista de hits (opcional)

## Observacoes

- Para cancelar, digite `c`.
- Se quiser ver detalhes, escolha "Mostrar lista detalhada".
