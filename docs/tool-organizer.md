# DevTools.Organizer - Como usar (CLI)

## Objetivo

Organizar documentos por categoria e deduplicar.

## Passos

1. Selecione `Organizer`.
2. Informe pasta de entrada e pasta de saida.
3. (Opcional) Informe config e min score.
4. Escolha se vai aplicar ou so simular.

## Entradas

- Pasta de entrada (obrigatorio)
- Pasta de saida (obrigatorio)
- Config (opcional)
- MinScore (opcional)
- Aplicar mudancas (s/n)

## Saida

- Estatisticas (mover, duplicados, ignorados, erros)
- Plano de acao (opcional)
