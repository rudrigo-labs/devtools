# DEFINICAO - Tutorial de Criacao de Ferramenta

- Status: Consolidada
- Criado em: 2026-03-07 02:36
- Objetivo: Definir um passo a passo oficial para criar novas ferramentas no DevTools sem perder padrao e rastreabilidade.

## Escopo
1. Definir fluxo unico de criacao de ferramenta.
2. Consolidar ordem de implementacao para evitar retrabalho.
3. Incluir rastreabilidade obrigatoria no processo.

## Regra principal
Toda nova ferramenta deve seguir a ordem:
1. Definicao da demanda.
2. Pendencias e fases.
3. Log cronologico.
4. Implementacao em Core/Tools/Infrastructure/Host no fluxo oficial.
5. Migrations no fechamento da integracao.
6. Validacao final (build/testes/evidencias).
