DevTools

Suite desktop em .NET 10 + WPF para acelerar tarefas de engenharia de software no Windows.

Repositorio Releases

O que e

Aplicacao unica em estilo IDE, com ferramentas tecnicas acopladas em uma experiencia consistente: notas, busca, renomeacao, snapshot, migracoes, SSH tunnel, ngrok e mais.

Stack

  • .NET 10
  • WPF / XAML
  • SQLite e JSON (configuravel)
  • GitHub Actions (CI)
  • Inno Setup (instalador)

Ferramentas principais

  • Notes (txt/md + backup + sync opcional Google Drive)
  • Organizer
  • Harvest
  • SearchText
  • Rename
  • Snapshot
  • Utf8Convert
  • Image Splitter
  • Migrations
  • SSH Tunnel
  • Ngrok
  • Jobs e Logs
