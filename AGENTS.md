# Regras do repositório

## Idioma
Todo texto deve ser escrito em português brasileiro (pt-BR) com acentuação correta.

Exemplos corretos:
configuração
validação
integração
informação
ação
não

Escrever palavras portuguesas sem acento é considerado erro.

## Exceção
Não usar acentos em identificadores técnicos:

- variáveis
- classes
- namespaces
- arquivos
- chaves JSON
