# Pacote Base de Regras e Documentacao

Este pacote foi extraido para reutilizacao em outros projetos.

## Conteudo
- Regras de agente em `.ai/`
- Indice raiz `AGENT_RULES_INDEX.md`
- Politica e controle de documentacao
- Templates de demanda/pendencias/log/checklist

## Uso rapido
1. Copiar estrutura para o novo projeto.
2. Ajustar `.ai/PROJECT_RULES.md` para o projeto de destino.
3. Manter `AGENT_RULES_INDEX.md` como indice principal.
4. Usar `docs/templates/` para abrir demandas com rastreabilidade.
